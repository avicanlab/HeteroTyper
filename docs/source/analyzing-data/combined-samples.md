# Lag time and other phenotypic features

`plot_combined_samples.m`

## Purpose

Plots per-group probability density histograms for user-selected phenotypic features
across all sample groups in the experiment. Each panel displays the distribution of one
feature for one group, with a median line and Gini index annotation. Up to 10 features
and any subset of groups can be plotted in a single figure. After rendering the figure,
the function also prints a biological replicate correlation and statistical analysis to
the terminal and exports it as an Excel workbook (one sheet per sample group) if
`ht.biorep_corr` was populated by `preprocess_pipeline_data`.

## Requirements

- MATLAB R2020a or later (`exportgraphics` required).
- `preprocess_pipeline_data(data);` must be run first to generate the `ht` struct.

## Usage

Pass the `ht` struct as the only argument. An interactive terminal prompt will ask for
group and feature selections before plotting begins. All other steps are automatic.

```matlab
plot_combined_samples(ht);
```

## Input

| Parameter | Description |
|---|---|
| `ht` | Master struct produced by `preprocess_pipeline_data`. Must contain `ht.groups`, `ht.params`, `ht.labels`, `ht.colors`, `ht.xbins`, and `ht.global`. If `ht.biorep_corr` is present (generated by STEP 6f of `preprocess_pipeline_data`), biological replicate statistics are printed and exported to Excel automatically. |

## Interactive selection

On launch, a menu is printed to the terminal showing all available groups and features.
Two selections are made in sequence:

- **Groups** → enter labels separated by spaces or commas (e.g. `3h 24h`), or type
  `ALL`
- **Features** → enter feature numbers separated by commas (e.g. `1 3 5`), or type
  `ALL`

The 10 available features are:

| Parameter | Feature |
|---|---|
| 1 | Lag time (h) |
| 2 | Colony size (px) |
| 3 | Area (px) |
| 4 | Intensity |
| 5 | Mean intensity |
| 6 | Intensity / size |
| 7 | Perimeter (px) |
| 8 | Circularity |
| 9 | Eccentricity |
| 10 | Solidity |

:::{note}
Intensity, mean intensity, and intensity / size histograms are recomputed locally with
finer bins (~100 bins) rather than the coarser precomputed bins stored in `ht.xbins`.
All other features use precomputed histogram counts directly.
:::

## Figure Layout

The figure is a grid of panels with one row per selected group and one column per
selected feature. All panels within a column share the same x-axis range. Each panel
shows:

- Filled bar histogram (probability density) in the group colour
- Red vertical median line
- Group label and Gini index in the top-left corner

Mirror tick marks are drawn on the top and right axes of every panel for a
publication-ready appearance.

## Outputs

Three figure files and one log are written to `ht.params.out_dir`, all with a
timestamp and feature-list suffix:

| Parameter | Description |
|---|---|
| `HeteroTyper_features_<feat>_<timestamp>.fig` | MATLAB figure — re-openable and editable |
| `HeteroTyper_features_<feat>_<timestamp>.pdf` | Vector PDF for publication |
| `HeteroTyper_features_<feat>_<timestamp>.png` | Raster PNG at 300 dpi |
| `plot_combined_samples_<timestamp>.txt` | Log file — selected groups, features, per-group value ranges, and file paths |
| `biorep_correlation_stats_<timestamp>.xlsx` | Excel workbook with biological replicate statistics. One sheet per sample group (pairwise Pearson r, Spearman rho (ρ), KS D, Jensen-Shannon Divergence (JSD), \|dGini\|, \|dMedian\|, CV% of medians) plus a per-replicate Gini sheet (`Gini_<group>`) and a README sheet. Only written when `ht.biorep_corr` is present and was not skipped. |

## Typical Workflow

```matlab
% Step 1 — run preprocessor once per session
preprocess_pipeline_data(data);

% Step 2 — plot phenotypic feature histograms
plot_combined_samples(ht);
```

At the prompt, enter groups and features. Example session:

- **Groups** → enter labels separated by spaces or commas (e.g. `3h 24h`), or type `ALL`
- **Features** → enter feature numbers separated by commas (e.g. `1 3 5`), or type `ALL`

## Biological Replicate Statistics

If `preprocess_pipeline_data` was run with a biological replicate column selected
(STEP 6f), `plot_combined_samples` automatically prints a full statistical comparison
table to the terminal and exports it to an Excel workbook. The following tests are
computed per replicate pair per morphology parameter:

| Metric | Description |
|---|---|
| Pearson r | Linear correlation of 20-quantile profiles across replicate pairs. r ≈ 1 = strong linear agreement in distribution shape. Two-tailed p-value (df=18). |
| Spearman rho (ρ) | Non-parametric rank correlation of the same 20-quantile profiles. Robust to skewed distributions. rho ≈ 1 = strong rank agreement. Two-tailed p-value (df=18). |
| KS D | Kolmogorov-Smirnov max-CDF distance [0, 1]. D ≈ 0 = near-identical distributions. KS p starred as `*** p<0.001`, `** p<0.01`, `* p<0.05`, `ns`. |
| JSD | JSD [0, 1]. JSD ≈ 0 = identical histogram shapes; JSD ≈ 1 = maximally different. |
| \|dGini\| | \|Gini_A − Gini_B\| on raw colony values. ≈ 0 = both replicates have similar within-replicate phenotypic heterogeneity. |
| \|dMedian\| | Absolute difference of per-replicate medians in original units. |
| CV of medians | CV% of per-replicate medians across all replicates in the group. ≤ 5%: excellent reproducibility; 5–15%: good; 15–30%: moderate; > 30%: high variability. |

If STEP 6f was skipped (no biological replicate column was selected during
preprocessing), the terminal prints "BIOLOGICAL REPLICATE STATISTICS — SKIPPED" with
instructions on how to enable the analysis by re-running `preprocess_pipeline_data`
with a replicate column. No Excel file is written in that case.

## Troubleshooting

| Error Message | Description |
|---|---|
| Group label not recognised | Labels are case-insensitive; the label just has to match one shown in the menu (e.g. `'24h'` or `'24H'`). |
| Empty / flat histogram for a feature | No valid values exist for that feature in that group. Check that the feature was measured during acquisition. |
| `exportgraphics` error | Requires MATLAB R2020a+. Use `print()` as a fallback on older versions. |
| Log file not created | Check that `ht.params.out_dir` exists and is writable. |
| "BIOLOGICAL REPLICATE STATISTICS — SKIPPED" printed; no Excel file | No biological replicate column was selected when `preprocess_pipeline_data` was run. Re-run `preprocess_pipeline_data` and at parameter step 6 select the metadata column identifying each biological replicate (e.g. "Set", "BioRep") — do NOT press 0. |
| "No biorep_corr field found" printed; no Excel file | The `ht` struct was produced by an older version of `preprocess_pipeline_data` that did not include STEP 6f. Re-run the updated `preprocess_pipeline_data(data)` to regenerate `ht` with the `biorep_corr` field populated. |
